# Asynchronous Server Technologies


## Introduction

This is the Node.js application project.

## Installing

```bash
git clone https://github.com/JeromeDltr/AsynchronousServerTechnologies.git
cd AsynchronousServerTechnologies
npm install
```

## Build

```bash
npm run build
```

## Development

```bash
npm run dev
```

## Database population

```bash
npm run populate
```
